== Description ==

Solpay Donation & Fundraising leverages the power of the Solana Blockchain to enable people to make donations instantly, with zeros costs. This free plugin uses Solana Pay and helps you set up a donations page on any WordPress site. With this free plugin you can start accepting donations in no time.
You install the plugin, set up your wallet and copy the [solpay_donation_form] shortcode wherever you want the form to be displayed. Try it out now, it's super simple and quick!
It works on mobile and desktop and we've set up some default donation buttons that you can use. Install on any WordPress site and start accepting donations from anyone with a Phantom wallet now!

== Installation ==

AUTOMATIC INSTALLATION

1. Log in to your WordPress dashboard, navigate to the Plugins menu and click "Add New". In the search field type "Solpay Donations & Fundraising" and click Search Plugins. After you've found our plugin install it by clicking "Install Now".

2. Enable the plugin.

3. Go to Settings -> Solpay Donations & Fundraising to update the settings and fill in your Solana wallet address.

4. Use the following shortcode: [solpay_donation_form] in any page or sidebar / footer widget to display the form.
